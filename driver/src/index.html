Steps:

  • Turn on Hioki device (press on button for 2sec)
  • Turn on bluetooth of Hioki device (Quick press the on button)
  • Turn on your computer's bluetooth
  • Search for the Hioki device (name should be something like LR8515....) and try to connect it to your computer using the bluetooth settings
  • Click connect button below
  • Select the Hioki device and click open. The app will now attempt to make a serial connection with the Hioki device.
  • Click the send button to send a few messages
  • Inspect response content in the text area below
Supported browsers*: Chrome >= 89, Edge >= 89 and Opera >= 75.
*Web Serial API is used to read and write data to a serial port. More info: https://web.dev/serial/
